## 🧭 Sobre mim

Trabalho com **Business Intelligence, Engenharia de Dados e Análise de Dados**, construindo soluções de ponta a ponta: da entrada e limpeza dos dados (ETL) até o modelo dimensional, as regras de negócio em DAX/SQL e a experiência visual final.

Meu projeto de conclusão de curso aplicou essa jornada completa a um problema institucional real do **Instituto Federal do Ceará (IFCE)**: transformar 133 mil matrículas de 33 campi em um dashboard de gestão acadêmica navegável por qualquer gestor, sem treinamento em BI.

Construí sozinha, do levantamento de requisitos à publicação, um pipeline de BI que hoje sustenta o acompanhamento de evasão de 133 mil matrículas do IFCE — desde a limpeza dos dados brutos em Power Query até as 70 medidas DAX e a interface final. Buscando minha próxima posição como **Analista de Dados / Desenvolvedora Power BI / Cientista de Dados**.

## 📌 Projeto em destaque

### 🎓 [Sistema de BI para Gestão Acadêmica — IFCE](https://github.com/jessicalimaxlsx/ifce_dashboard_academico)

Pipeline completo de Business Intelligence cobrindo entrada de dados, ETL, modelo dimensional e UX/UI, aplicado a 133 mil matrículas de 33 campi do IFCE.

- Pipeline de ETL em **Power Query** com tipagem, limpeza e validação de dados
- Modelo em **Star Schema** com medidas **DAX** para padronizar regras de negócio (evasão, conclusão etc.)
- Dashboard de 24 páginas com identidade visual própria, desenhada do zero no PowerPoint
- 🔗 [Dashboard ao vivo](https://app.powerbi.com/view?r=eyJrIjoiNjNkY2MwNjMtOGNiZi00NmRjLWFlN2YtOGVlMjYxNjgyNGIxIiwidCI6IjI0YjcyZmM0LWNhZWMtNDllMi1iMGE3LTcyNjY3NTQxYWE5MyJ9) · [Código](https://github.com/jessicalimaxlsx/ifce_dashboard_academico)
